Sobre o projeto

Bem-vindo ao Alugaí, sua plataforma ideal para alugar tudo o que você precisa! Alugaí é uma plataforma web desenvolvido em Spring MVC, projetado para fornecer aos usuários uma maneira conveniente e eficiente de alugar uma ampla variedade de itens.

Navegue e pesquise

Navegue facilmente por nossa coleção diversificada de itens disponíveis para aluguel ou use a funcionalidade de pesquisa para encontrar itens específicos.

Processo de aluguel

Alugar um item na Alugai é rápido e simples. Depois de encontrar o item que deseja alugar, basta selecionar o período de locação e prosseguir para a finalização da compra.

Contas de usuário

Crie uma conta personalizada no Alugaí para gerenciar seus aluguéis, acompanhar seu histórico de pedidos e salvar seus itens favoritos para referência futura.

Pagamentos seguros

Priorizamos a segurança de suas transações. Alugaí oferece processamento de pagamentos seguro para garantir que suas informações financeiras estejam protegidas.

Avaliações e comentários

Ajude outros usuários a tomar decisões informadas, deixando avaliações e comentários sobre os itens que você alugou.

Design responsivo

Nosso site foi projetado para funcionar perfeitamente em desktops, tablets e dispositivos móveis, garantindo uma experiência tranquila ao usuário, independentemente de como você acessa o Alugaí.

Tecnologias utilizadas

Alugaí é construído utilizando as seguintes tecnologias:

  • Spring MVC: Uma estrutura poderosa para construir aplicativos da web baseados em Java.
  • HTML/CSS/JavaScript: tecnologias frontend para criar uma interface de usuário responsiva e interativa.
  • MySQL: Um sistema de gerenciamento de banco de dados relacional usado para armazenar dados de usuários e itens.
  • Hibernate: Uma estrutura ORM (Mapeamento Objeto-Relacional) para mapear objetos Java para tabelas de banco de dados.
